Environmental Pollution Home

  • Computer Graphics-Mini Project

  • Team Size:2

  • Software Requirements:

    • • Programming Language: C/C++
    • Library: OpenGL
    • IDE: Codeblocks

  • Description:

    • The problem statement for this project is to develop a computer graphics simulation that visually depicts different types of environmental pollution and its sources. By creating an interactive and immersive environment, we aim to provide users with a firsthand experience of the detrimental effects of pollution and increase awareness about the need for sustainable practices.
